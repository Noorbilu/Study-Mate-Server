This is the secure backend server for the StudyMate app.
It's built with Node.js, Express.js, and MongoDB to store and manage all app data.
Main Features:
CRUD Operations: Handles create, read, update, and delete requests.
Search & Sort: Finds and organizes study partners based on user preferences.
Auto Count: Automatically increases the partner connection count when a request is sent.
